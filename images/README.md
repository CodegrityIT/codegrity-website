# Images Folder

## How to Add Your Images

### Team Photos
1. Take professional headshots of your team
2. Save as: `teamname.jpg` (e.g., `john-doe.jpg`)
3. Put in `/images/team/` folder
4. Update `about.html` to reference the image

### Project Screenshots
1. Take screenshots of your projects
2. Save as: `project-name.jpg`
3. Put in `/images/projects/` folder
4. Update portfolio page when you create it

### Other Images
Any other images go in `/images/misc/`

## Recommended Sizes
- Team photos: 600x600px (square)
- Project screenshots: 1200x800px (landscape)
- Logos: Use SVG files in `/assets/logos/`

## Optimize Before Uploading
Use https://tinypng.com/ to compress images before uploading.
Target: Keep images under 200KB each.
